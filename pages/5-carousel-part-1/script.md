# Step

Customize the ease

```jsx
ease: [0.32, 0.72, 0, 1];
```

```
<motion.div
  animate={{ x: `-${index * 100}%` }}
  transition={{
    duration: 0.7,
    ease: [0.32, 0.72, 0, 1],
  }}
  className="flex"
>
```
